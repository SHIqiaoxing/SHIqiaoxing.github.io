Project Idea

Overview

Compared with our impression of full screen character programming, low code development is a way of program development without code or using less code. We can compare the use of low code development to LEGO. Each functional module is like a small building block. If we put them together in different ways, we can get different finished products.

Motivation

Generally speaking, in the process of developing a program, a considerable part of the work of programmers is repetitive and cumbersome. According to the data of frevvo, low code development can increase the speed of digital transformation by 69%, which can not only reduce the dependence on high-tech level, but also shorten the development time of applications, Programmers can be liberated from this part of the work, and more people can participate in application development. Not only programmers with professional programming ability, but also business personnel with non-technical background can build applications. For large enterprises, low code development platform can also reduce the initial cost of it team training and technology deployment.

Description

Fast start: due to the characteristics of low code, it will undoubtedly greatly reduce the difficulty of learning programming language, especially the 0 code development platform, and even business personnel who do not understand programming language can quickly learn and develop applications.

Fast development: due to the use of a large number of components and encapsulated interfaces for development, as well as the IAAs and PAAS layer capabilities of integrated cloud computing, the development efficiency is greatly improved; the general view is that low code can improve the development efficiency by more than 30%, while 0 code (no code) can improve the development efficiency several times; and greatly reduce the development cost.

Fast running: This is a relative concept. Generally speaking, due to the low code development platform using automatic compilation into executable code, the overall quality of the code is better than the average level of the industry; and relatively speaking, bugs are more controllable, and the security of the code will be higher.

Fast operation and maintenance: in general, low code development platform adopts component form and object-oriented development mode, which makes the code more structured and easier to maintain.

Low code refers to a development method, more specifically, a development platform, which is called "low code development platform". Specifically, it refers to a development platform that can quickly generate applications without coding or through a small amount of code. It is a method of application development through visualization, so that developers with different levels of experience can go through the graphical user interface Using drag and drop components and model driven logic to create web pages and mobile applications.

The main user groups of this product are a large enterprise company and a non R & D information internet company. Through the low code platform, enterprises can build their own information management platform according to their own needs, reduce the cost of information transmission between different levels and departments, and facilitate the collection and integration of information, which can help enterprises quickly realize digitization transformation.
